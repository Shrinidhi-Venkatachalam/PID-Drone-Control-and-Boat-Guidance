# PID-Drone-Control-and-Boat-Guidance
# Drone Guidance and Control Using Python Simulations

## Overview

This project demonstrates the fundamentals of Guidance and Control systems used in autonomous vehicles such as drones, rockets, and marine vehicles. The simulations were implemented in Python using Google Colab and focus on understanding feedback control, PID controller tuning, disturbance rejection, and trajectory tracking.

The project consists of two main simulations:

### 1. Drone Altitude Control

A PID controller is used to control the altitude of a drone and maintain a desired height. Different values of Proportional (P), Integral (I), and Derivative (D) gains were tested to analyze system response, overshoot, stability, and disturbance rejection under simulated wind conditions.

### 2. Autonomous Boat Path Tracking

A guidance algorithm was implemented to enable an autonomous boat to follow a predefined path. The simulation demonstrates trajectory tracking, feedback correction, and navigation performance under varying environmental conditions.
